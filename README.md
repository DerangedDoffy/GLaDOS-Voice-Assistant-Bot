# GLaDOS Voice Assistant Bot 

## **Install**
* pip install pyaudio
* pip install playsound 
* pip install gTTS
* pip install datetime 
* pip install pyautogui
* pip install wheel 

**Go to: https://www.lfd.uci.edu/~gohlke/pythonlibs/#pyaudio if you have trouble installing Pyaudio & install .whl**

## **Commands Available**
* Hello 
* Name
* Time
* Date
* Search 
* Screenshot
* Sneeze *you heard that*
* Reference Joke
* Open Apps
* Wish Merry Xmas
* Goodbye
* Exit
* Small voicelines etc
